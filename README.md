# Mark Adjustments Dashboard (2025)

Project Overview

Mark adjustments are required when a student's final mark needs to be manually amended on the university system. These adjustments may arise due to data capture errors, calculation discrepancies, system-related issues, approved academic interventions, or other exceptional circumstances requiring administrative correction.

To improve monitoring and reporting, I developed an interactive dashboard that consolidates all mark adjustment requests submitted during 2025. The dashboard provides management and administrative staff with a clear overview of submission trends, faculty activity, and the status of requests.

Dashboard Features
Monthly Mark Adjustment Trends

The line chart displays the number of mark adjustment requests received each month during 2025. This visualization helps identify peak submission periods, allowing departments to anticipate workloads and allocate resources more effectively.

Faculty Analysis

The bar chart illustrates the total number of mark adjustment requests submitted by each faculty during 2025. This enables comparisons across faculties and highlights areas where additional support, training, or process improvements may be required.

Request Status Monitoring

The mark adjustment process requires a correctly completed form together with all relevant supporting documentation. Requests that are incomplete or contain errors are referred back to the originating faculty for correction and resubmission.

A slicer allows users to filter and view requests that were referred back. This provides insight into the quality of submissions received and helps identify faculties that may require additional guidance or training on the mark adjustment process.

Key Insights

Mark adjustment requests were highest during February and December, indicating peak administrative periods.
Arts and Humanities submitted the highest number of mark adjustment requests during 2025.
The dashboard enables monitoring of requests that were referred back due to incomplete documentation or incorrectly completed forms.
Trend analysis assists with workload planning and resource allocation during peak periods.
Referred-back requests can be used as an indicator of process compliance and training needs across faculties.
